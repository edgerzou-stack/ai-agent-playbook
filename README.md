# AI Agent Playbook

> **The Ultimate Methodology for Human-AI Collaboration**

本仓库致力于记录和沉淀与高级 AI Agent 协作时的最高效范式、底层架构哲学以及核心 SOP（标准操作程序）。

## 核心哲学 (Core Philosophy)

1. **Code Logic First (代码逻辑优先)**
   任何对于系统的描述与文档，必须严格锚定底层真实的算法、代码逻辑和操作算子。拒绝一切“想当然”的过度拟人化和脱离实际的模糊描述。
2. **纯净极客美学 (Geek Minimalism)**
   排版追求绝对的高信噪比：拒绝表情符号 (Emojis)、花哨标签 (Badges)，一切以结构化文本、列表、表格和 Mermaid 流程图为最高优先级。
3. **消除黑盒 (Eliminate Blackbox)**
   Agent 必须处于“完全可观测状态”。状态的流转、记忆的读取、计划的变更必须对用户保持 100% 透明。
